# jira-filters-automation
Selenium WebDriver (Java) automation for validating Jira Filters workflow, including creation and verification of Open and Closed/Done issue filters, with documented assumptions and limitations.

# Objective
This project demonstrates a UI automation approach for validating the *Jira Filters workflow* using Selenium WebDriver with Java.

The automation covers:
 Open Items Filter
 Closed/Done Items Filter

# Tech Stack
 Selenium WebDriver
 Java
 TestNG
 Maven
